## Installation
To install, use [npm](https://www.npmjs.com/) :
```bash
# Install dependencies
npm install
# Start local dev server
npm run dev
```

To deploy on Github Pages :
```bash
# Publish dist folder to gh-pages branch
npm run deploy
```
